# 🏦 بوت البنك - Discord Bank Bot

بوت ديسكورد متكامل لنظام اقتصادي داخل السيرفر مع أكثر من **80 أمر** وردود على شكل **صور مولّدة**.

## ✨ المميزات

- 💰 **نظام اقتصادي كامل** - رصيد، بنك، تحويل، راتب
- 💼 **نظام وظائف** - 10 وظائف بمستويات قابلة للترقية
- 📈 **استثمار وأسهم** - أسعار تتغير كل 10 دقائق
- 🏠 **عقارات** - 9 أنواع من الغرفة للجزيرة
- 🎮 **ألعاب** - عملة، نرد، سلوت، بلاك جاك، روليت
- 🥷 **جرائم** - سرقة، سطو، تهريب مع نظام سجن
- ⛏️ **موارد** - صيد، تعدين، احتطاب، زراعة
- 🐾 **حيوانات** - 6 أنواع بمستويات
- ⚔️ **نقابات** - إنشاء، دعوة، حروب
- 🏆 **إنجازات ومهام** - نظام XP ومستويات
- 💑 **زواج** - نظام عائلي
- 🎉 **أحداث** - يانصيب، سباق، مزاد
- 🖼️ **كل الردود صور** - لا embed واحد!

## 🚀 التشغيل على Railway

1. اضغط **Fork** على هذا المستودع أو **Deploy on Railway**
2. في Railway، أضف متغير البيئة:
   ```
   DISCORD_TOKEN=توكن_بوتك
   ```
3. Railway سيشغل `node src/index.js` تلقائياً

## ⚙️ الإعداد بعد التشغيل

بعد إضافة البوت للسيرفر، حدد قناة البنك:

```
/set channel bank #اسم-القناة
```

أو بدون منشن، سيحدد القناة الحالية تلقائياً.

## 📋 الأوامر الكاملة

### 💰 الاقتصاد
| الأمر | الوصف |
|-------|-------|
| `رصيدي` / `فلوسي` / `محفظتي` | عرض رصيدك |
| `ايداع [مبلغ]` | إيداع في البنك |
| `سحب [مبلغ]` | سحب من البنك |
| `تحويل @عضو [مبلغ]` | تحويل لعضو |
| `راتب` | استلام راتب كل 3 ساعات |
| `اسبوعي` | مكافأة أسبوعية |
| `شهري` | مكافأة شهرية |
| `مكافأة` | مكافأة يومية |
| `ثروتي` | عرض الثروة الكاملة |
| `اغنى` / `توب` | أغنى 10 لاعبين |

### 💼 الوظائف
`وظائف` • `توظف [وظيفة]` • `عملي` • `استقل` • `ترقية` • `دوام`

### 📈 الاستثمار
`استثمار [مبلغ]` • `استثماراتي` • `بيع_استثمار [رقم]` • `اسهم` • `شراء_سهم [رمز] [كمية]` • `بيع_سهم [رمز] [كمية]` • `سوق`

### 🏠 العقارات
`عقارات` • `شراء_عقار [اسم]` • `بيع_عقار [اسم]` • `عقاراتي` • `تحصيل_ايجار`

### 🎮 الألعاب
`عملة [وجه/كتابة] [مبلغ]` • `نرد [مبلغ]` • `سلوت [مبلغ]` • `بلاك_جاك [مبلغ]` • `روليت [مبلغ]` • `تخمين [1-10] [مبلغ]` • `حظ`

### 🥷 الجرائم
`جريمة` • `سرقة @عضو` • `سطو @عضو` • `تهريب` • `اختلاس` • `هروب`

### ⛏️ الموارد
`صيد` • `تعدين` • `احتطاب` • `زراعة` • `جمع` • `تنقيب`

### 🛒 المتجر
`متجر` • `شراء [غرض]` • `بيع [غرض]` • `حقيبتي` • `استخدام [غرض]` • `اغراضي`

### 🐾 الحيوانات
`حيوانات` • `شراء_حيوان [اسم]` • `حيواني` • `اطعام` • `لعب` • `تطوير_حيوان`

### ⚔️ النقابات
`انشاء_نقابة [اسم]` • `نقابتي` • `دعوة_نقابة @عضو` • `طرد_نقابة @عضو` • `حرب_نقابات`

### 🔧 الإدارة (مشرفين فقط)
`اضافة_فلوس @عضو [مبلغ]` • `خصم_فلوس @عضو [مبلغ]` • `تصفير @عضو` • `اعطاء_غرض @عضو [غرض]` • `سحب_غرض @عضو [غرض]`

## 🏗️ التقنيات المستخدمة

- **Discord.js v14**
- **@napi-rs/canvas** - توليد الصور
- **better-sqlite3** - قاعدة البيانات
- **Node.js 18+**
